This directory contains the frontend for our Space Station Safety Object Detection system, built using Lovable AI UI Builder and connected to the model-powered Streamlit backend for real-time inference.
The frontend provides a simple, user-friendly interface to interact with the detection model without requiring users to run code manually.

## Features
Fully responsive UI built using Lovable
Integrated with the Streamlit inference backend
Allows users to run detection through browser interface
Supports image upload and camera input for real-time testing
Displays bounding boxes, confidence scores, and detected class names
